### Swiftbot API | RESTful Backend for Discord Bot

A fully-featured RESTful API built with **Node.js**, **Express.js**, and **MongoDB**, connecting the Swiftbot Dashboard to its Discord bot database. Designed to manage users, servers, and bot features via a secure, interactive backend.

**Key Systems & Features:**
- **Banking System:**
  - Create/delete account
  - Transfer balance to others
  - Daily rewards
  - Change account PIN
  - Show account details (account name, account type, last reward claimed date, next tax date, previous tax date, account balance, etc..)
  - Transaction history (recipient, amount, reason, date, etc...)
  - Privacy settings (show balance for others, and toggle require account PIN on dashboard)
- **Marketplace System:**
  - Purchase rank cards & profile cards related to the bot
  - Create/modify/delete rank/profile cards available on the marketplace (admins only)
  - Equip or change an equipped rank/profile card
  - See all owned cards
- **Greeting System:**
  - Enable/disable the system
  - Setup/modify greeting message, greeting image, and the greeting channel
  - Delete the system
  - Get the current system's configurations
- **Leveling System:**
  - Enable/disable the system
  - Setup/modify the level-up message and the channel where it will be sent at
  - Delete the system
  - Get the current system's configurations
- **Auto-Responses System:**
  - Enable/disable the system 
  - Create/modify/delete a auto-response
  - Get all available auto-responses
  - Delete all auto-responses at once
- **Ticketing System:**
  - Setup/delete the system
  - Modify the ticketing category and/or the transcripts channel
  - Send a ticket panel
- **Giveaways System:**
  - Start a new giveaway (while specifying the winners count, duration, prize, and the channel where the giveaway will start)
  - Modify/end an active giveaway
  - Reroll an ended giveaway

**Technologies:**
- Node.js & Express.js
- MongoDB with Mongoose
- JWT Authentication
- CORS & dotenv
- Middleware for authentication & permissions
